# 🛍️ Clone de Boutique en Ligne développement web

Bienvenue sur mon projet personnel de développement web !

Dans ce projet, j'ai recréé le site web de la boutique en ligne **svapo**, renommé ici **Sviluppo**, dans le but d'explorer les fondamentaux du développement web et d'améliorer mes compétences techniques. (Veut dire developpement web) Ce projet est une initiative pédagogique visant à comprendre la structure d'un site e-commerce moderne.

---

## 🎯 Objectifs du projet

- Reproduire une boutique en ligne fonctionnelle
- Expérimenter avec les technologies front-end et back-end
- Apprendre à créer une interface utilisateur responsive
- Simuler une expérience d'achat réaliste
